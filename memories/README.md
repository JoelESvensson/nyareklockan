# Memories / Minnen från 2025

Lägg till bilder och videor från 2025 här. Filerna visas automatiskt under den sista timmen innan midnatt.

## Filnamnformat

Filnamnet ska börja med månadens namn (på svenska eller engelska):

- `januari.jpg` → "Januari, 2025"
- `february-skiing.mp4` → "Februari, 2025"  
- `mars_semester.jpg` → "Mars, 2025"
- `juli-strand-semester.mp4` → "Juli, 2025"

## Stödda format

- **Bilder:** `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
- **Video:** `.mp4`, `.webm`

## Exempel på filnamn

```
januari.jpg
februari-alla-hjartans-dag.jpg
mars-skidresa.mp4
april.png
maj-valborg.jpg
juni-midsommar.mp4
juli-semester.jpg
augusti-kräftskiva.jpg
september-höstpromenad.jpg
oktober-halloween.mp4
november.jpg
december-jul.jpg
```

## Tips

- Videor spelas upp utan ljud (muted)
- Varje minne visas i 10 sekunder
- Minnena visas en gång per minut under sista timmen
- Under de sista 10 minuterna slutar minnena visas och det dramatiska läget tar över
